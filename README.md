# QuackJS
🦆 Что такое quackjs  quackjs — это клиентская JS-библиотека, которая:  подключается к серверу (например, по WebSocket);  получает новые уведомления в реальном времени;  показывает их пользователю (в виде всплывающих тостов и/или системных уведомлений);  позволяет слушать события (notification, connect, error, и т.д.).
